POLITIQUE ANTI-FRAUDE IMMOBILIÈRE

Date de dernière mise à jour : 02/05/2026

Version : 1.1

Afin de lutter contre les arnaques immobilières, Kërjom applique une politique stricte.

1. Vérification des annonceurs

Les professionnels peuvent être soumis à une vérification :

  • identité
  • registre de commerce
  • documents professionnels
  • NINÉA

2. Détection des annonces suspectes

Kërjom peut analyser :

  • prix anormalement bas
  • photos volées
  • descriptions trompeuses
  • comptes multiples.

3. Signalement par les utilisateurs

Tout utilisateur peut signaler :

  • une arnaque
  • une annonce suspecte
  • un comportement frauduleux.

4. Mesures en cas de fraude

Kërjom peut :

  • supprimer l’annonce
  • suspendre le compte

transmettre les informations aux autorités.
